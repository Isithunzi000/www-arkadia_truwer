# arkadia_truwer

Chrome extension - asystent odgrywania scen dla oficjalnego klienta [arkadia.rpg.pl](https://arkadia.rpg.pl).

## Instalacja

1. Pobierz i rozpakuj `arkadia_truwer.zip`
2. Chrome: `chrome://extensions/` → tryb dewelopera → zaladuj rozpakowany folder
3. Przejdz na `arkadia.rpg.pl`

## Uzycie

```
/truwer              - otworz/zamknij
/truwer float        - tryb plywajacy
/truwer left         - dok lewy
/truwer right        - dok prawy
/truwer help         - pomoc
```

## Aktualizacja

Po pojawieniu sie powiadomienia o nowej wersji: pobierz ZIP, rozpakuj do tego samego folderu, odswiez rozszerzenie w Chrome.
